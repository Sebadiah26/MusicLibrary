Import artists, albums & songs from CSV — then curate genre, sub-genre, favorites and ratings.

Import CSV

Expected columns
  • Artists: Name (required), Genre, SubGenre, IsFavorite, Rating
  • Albums: Artist (required), Album/Title (required), Year
  • Songs: Artist (required), Song/Title (required), Album, Duration (sec or mm:ss)

Header names are matched case-insensitively and ignore spaces. Unknown columns are ignored.

iTunes XML Import

About iTunes XML import

Upload your iTunes Music Library.xml file (found in your iTunes/Music folder). Tracks are parsed and imported as artists, albums, and songs. Use "Download as CSV" to get three CSV files compatible with the manual import above.
